# 概要
健康管理サイトを作成しました。

健康意識を高めることが当サイトの目的です。

ユーザーが記事を投稿し、他のユーザーがその記事を簡単に閲覧できて、いいねを付けれたり記事の投稿者をお気に入りのユーザーとして登録することができます。

# 使い方
記事の投稿、編集、削除。プロフィールの編集。いいね、お気に入り機能。条件に応じた称号獲得機能があります。
